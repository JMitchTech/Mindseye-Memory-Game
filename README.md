# 👁️ Mind's Eye — Memory Card Game

## Setup & Run

1. Install dependencies:
```
pip install flask
```

2. Run the app:
```
python app.py
```

3. Open your browser to:
```
http://localhost:5000
```

## How to Play
- Click any card to flip it
- Flip a second card — if they match, they stay revealed
- If they don't match, both flip back after a short delay
- Clear all pairs to win
- Three difficulty levels: Easy (8 pairs), Medium (12), Hard (16)
- Timer and move counter track your performance
